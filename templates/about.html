About the Project

This Alzheimer’s Risk Screening Tool is a machine learning-based web application that helps identify early signs of Alzheimer’s disease using structured clinical data.

It uses features such as MMSE scores, brain volume ratios, and demographic inputs to assess the probability of early dementia using a Random Forest classifier.

This tool is intended for educational and demonstrational use only, and should not be used as a substitute for professional medical diagnosis.

Built with ❤️ using Python, Flask, HTML/CSS, and scikit-learn.
